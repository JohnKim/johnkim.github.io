## 🚀 개발 블로그

Gatsby 를 기반으로 한 블로그 페이지로 Github Pages 로 배포하여 서비스 되도록 구성하였습니다.

## 실행하기

### develop 모드 실행

개발모드로 localhost 로 실행한다.

```shell
yarn install
yarn develop
```

### 배포하기

빌드하여, gh-pages 브랜치에 업로드 한다.

```shell
yarn deploy
```
